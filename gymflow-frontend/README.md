# GymFlow Frontend

GymFlow Frontend es la primera fase de una propuesta web pensada para gimnasios.  
El objetivo de esta etapa es ofrecer una presencia digital más profesional, centralizar información importante del negocio y facilitar el proceso inicial de nuevos clientes mediante un preregistro en línea.

## Demo en línea

[Ver demo](https://gymflowcr.com/)

## Objetivo del proyecto

Esta primera fase está enfocada en construir una solución simple, útil y escalable para gimnasios.  
La propuesta busca que un gimnasio pueda contar con:

- una landing page profesional
- información clara del negocio
- horarios, planes y beneficios visibles
- ubicación y contacto en un solo lugar
- formulario de preregistro para nuevos clientes

La idea es que esta base pueda crecer después hacia una solución más completa con backend, base de datos y módulos adicionales.

## Alcance actual

Actualmente el proyecto incluye:

- página principal informativa
- diseño responsive
- sección de planes, beneficios, galería y contacto
- integración visual de ubicación con Google Maps
- formulario de preregistro
- demo funcional del flujo de envío

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript
- Google Maps Embed
